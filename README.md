# PocketNAS

A lightweight personal NAS dashboard built on a Raspberry Pi Zero.

## Features
- Web-based file browser
- Storage monitoring
- Remote access with Tailscale
- Mobile-friendly interface

## Hardware
- Raspberry Pi Zero
- USB Hub
- Ethernet Adapter (Not needed if have Zero 2 W)
- External USB storage

## Roadmap
- File downloads
- Photo gallery
- Uploads
- Search
